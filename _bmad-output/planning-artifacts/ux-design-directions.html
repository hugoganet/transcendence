Jargon Tooltip

A blockchain stores data in blocks ℹ️ that are linked together using cryptographic hashes.

Block
A container of data on a blockchain. Each block holds a batch of transactions and a link to the previous block.
💡 Think of it like a page in a ledger book — each page records transactions and is numbered in order.
